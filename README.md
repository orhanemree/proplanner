Proplanner is easy way to prepare a weekly schedule. You can export your schedule as a PDF file!

## Technologies
* Frontend: Tailwind CSS + Vue.js
* Deployment: Vercel

## Development Setup
```terminal
$ git clone https://github.com/orhanemree/proplanner.git
$ cd proplanner
$ npm i
$ npm run dev
```
